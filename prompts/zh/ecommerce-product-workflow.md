# AI 电商商品图工作流：从参考图到主图、详情页与广告素材

一套商品图不是“同一个 Prompt 多生成几次”。更稳定的做法是：先锁定商品事实，再让每张图只承担一个销售任务，最后按统一标准检查整组素材。

## 1. 先准备商品 Brief

| 项目 | 要写清楚的内容 |
| --- | --- |
| 商品 | 名称、品类、实际用途、目标用户 |
| 不能改变 | 外形、轮廓、长宽比例、颜色、材质、标签区域、Logo 位置、开口/按键/接口、配件数量 |
| 本张任务 | 白底主图、场景图、细节图、卖点图或广告图，只选一个 |
| 可验证卖点 | 只写包装、说明书或品牌资料可以证明的内容 |
| 输出规格 | 平台、画幅、留白方向、是否允许文字 |

## 2. 复制这段“商品保真锁”

```text
以我上传的商品参考图为唯一商品事实来源。必须保持商品的整体外形、轮廓、长宽比例、真实颜色、材质质感、标签区域、Logo 位置、开口/按键/接口位置和配件数量。不得重新设计商品，不得添加参考图中不存在的配件、文字、认证、功能或装饰。允许改变的只有背景、布光、镜头角度和画面构图。
```

如果允许改变角度，也要限定在参考图可以合理推断的范围内。看不到的背面结构，不要让模型自由补全。

## 3. 按销售任务规划素材顺序

1. **白底主图**：让用户一眼确认“卖的是什么”。
2. **核心场景图**：展示商品在真实使用环境中的尺度和用途。
3. **材质细节图**：只放大一个可见细节，证明质感或工艺。
4. **单一卖点图**：一张图解释一个经过验证的卖点。
5. **尺寸或结构图**：数据必须来自真实资料，不让模型编造。
6. **广告素材**：在商品事实稳定后，再增加氛围、留白和营销构图。

并非每个商品都需要六张。宁可保留三张任务明确的图片，也不要凑出一组重复画面。

## 4. 四个可直接改写的 Prompt

### 白底商品主图

```text
[粘贴商品保真锁]

生成一张电商白底商品主图。商品完整可见，三分之四视角，主体居中，占画面约 78%，边缘清晰，接触阴影自然，背景为纯净不透明白色。使用柔和棚拍布光，准确呈现真实颜色和材质，不添加道具、文字、边框、水印或额外配件。画幅 [1:1]。
```

### 核心使用场景图

```text
[粘贴商品保真锁]

把商品放入 [真实使用场景]，用 [目标用户] 的视角展示 [一个核心用途]。商品仍是画面唯一主角，环境只提供尺度和使用语境，不遮挡商品，不增加未提供的配件。光线方向与商品一致，接触关系真实，为右侧文案保留干净留白。画幅 [4:5]。
```

### 材质细节图

```text
[粘贴商品保真锁]

生成一张商品材质细节特写，只展示参考图中真实可见的 [材质/工艺/结构]。使用微距产品摄影，细节锐利，纹理自然，光线克制，不夸大质感，不改变颜色，不生成参考图中不可见的内部结构。画幅 [1:1]。
```

### 单一卖点图

```text
[粘贴商品保真锁]

围绕经过验证的卖点“[卖点]”设计一张电商卖点图。用构图和真实使用情境表达卖点，不依赖虚构数据或夸张效果。商品完整、准确、清晰，为 [左侧/右侧] 标题保留留白。除我提供的文字外，不生成其他文字、数字、认证标识或图标。画幅 [4:5]。
```

## 5. 出图后做四道 QA

### 商品一致性

- 外形、比例、颜色、材质、标签和 Logo 位置是否与参考图一致？
- 按键、接口、开口和配件数量是否凭空变化？

### 构图与销售任务

- 这张图是否只完成一个任务？
- 商品是否足够突出，留白是否真的可用于排版？

### 文字与事实

- 是否出现乱码、伪 Logo、虚构认证或无法证明的性能数据？
- 如果文字不可靠，先输出无字底图，再用设计工具排版。

### 系列一致性

- 同一组图片里的商品颜色、比例和配件是否稳定？
- 光线、背景和视觉语气是否像同一个品牌，而不是六套模板？

## 6. 常见失败与修正

| 失败 | 不要这样修 | 更有效的修正 |
| --- | --- | --- |
| 商品被重新设计 | 继续追加“真实、高级” | 重申保真锁，并列出具体错误部位 |
| 背景抢主体 | 添加更多氛围词 | 删除无关道具，限制背景色与景深 |
| 包装文字乱码 | 要求“文字更清晰” | 生成无字底图，后期加入准确文字 |
| 一组图像不同商品 | 每张从头写 Prompt | 固定同一参考图、保真锁和色彩基准 |
| 卖点像虚假广告 | 加强冲击力 | 删除无法验证的数据和认证，只表现真实用途 |

## 这个库解决什么问题

- 不只给形容词，而是把主体、构图、光线、材质、背景、画幅和限制条件写进模板。
- 对商品图明确“允许改变什么、必须保持什么”，减少产品变形、Logo 漂移和包装文字幻觉。
- 先提供能直接复制的基础版本，再给变量和失败修正方向，方便快速迭代。
- Markdown 适合人工浏览，JSON 适合检索、脚本和应用集成。

## 使用方式

1. 先选择最接近任务的提示词，不要从空白开始堆形容词。
2. 替换方括号中的产品、服装、场景、颜色、画幅或标题。
3. 上传参考图时，单独列出不能改变的外形、比例、颜色、材质、标签和配件数量。
4. 每轮只修一个明显问题，并检查商品、文字、事实和系列一致性。
